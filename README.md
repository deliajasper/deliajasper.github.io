### Deploying to GitHub Pages

1. **Create a new repository on GitHub**
   - Go to https://github.com/new
   - Name it `yourusername.github.io` (replace `yourusername` with your actual GitHub username)
   - Make it public
   - Don't initialize with README (we'll push our existing files)

2. **Push your code to GitHub**
   ```bash
   # Initialize git in your project folder
   git init
   
   # Add all files
   git add .
   
   # Make your first commit
   git commit -m "Initial portfolio commit"
   
   # Add your GitHub repository as remote
   git remote add origin https://github.com/yourusername/yourusername.github.io.git
   
   # Push to GitHub
   git branch -M main
   git push -u origin main
   ```

3. **Enable GitHub Pages**
   - Go to your repository on GitHub
   - Click on **Settings** â†’ **Pages** (in the left sidebar)
   - Under "Source", select **main** branch
   - Click **Save**
   - Your site will be live at `https://yourusername.github.io` in a few minutes! ðŸŽ‰

## ðŸ“ Customization

### Personal Information

Edit `index.html` and replace the following placeholders:

- **Name**: Search for "Your Name" and replace with your actual name
- **Email**: Replace `hello@example.com` with your email
- **GitHub**: Replace `yourusername` with your GitHub username
- **LinkedIn**: Replace the LinkedIn URL with your profile
- **Resume**: Add your resume PDF and update the link

### Content

- **Hero Section**: Update the main headline and subtitle
- **Projects**: Edit the project cards to showcase your actual work
- **About Section**: Write your own story!
- **Skills**: Update the skills list with your expertise

### Colors

Want to change the color scheme? Edit these CSS variables in `index.html`:

```css
:root {
    --bg-main: #fef8f3;          /* Main background */
    --bg-secondary: #fff5ed;     /* Secondary background */
    --bg-card: #ffffff;          /* Card background */
    --text-primary: #2d1810;     /* Main text */
    --text-secondary: #7a5c4e;   /* Secondary text */
    --accent-coral: #ff6b6b;     /* Coral accent */
    --accent-teal: #4ecdc4;      /* Teal accent */
    --accent-yellow: #ffe66d;    /* Yellow accent */
    --accent-purple: #a167ff;    /* Purple accent */
}
```
